# CLI 契约：`trt` 子命令（文本替换工具）

**Feature**: 001-text-replace-tool | **Date**: 2026-06-01

定义 `cct trt`（别名调用，等价于 `cct text-replace-tool`）的命令行契约。

---

## 用法

```text
cct trt -d <目录> -o <被替换内容> -n <替换内容> [选项]
cct trt -d <目录> --rules <规则文件> [选项]
cct trt -d <目录> -u 1 [选项]
```

## 参数表

| 参数 | 短选项 | 长选项 | 描述 | 默认值 | 对应需求 |
|------|--------|--------|------|--------|----------|
| directory | `-d` | `--directory` | 要处理的目录（相对或绝对路径） | 必填 | FR-006 |
| old_text | `-o` | `--old-text` | 要替换的文本（撤销模式可选） | 见下 | FR-007 |
| new_text | `-n` | `--new-text` | 替换后的文本（撤销模式可选） | 见下 | FR-007 |
| rules | | `--rules` | 多组替换规则文件（每行一组，制表符分隔） | 可选 | FR-007a |
| backup | `-b` | `--backup` | 启用备份（1=启用, 0=禁用） | 1 | FR-010 |
| undo | `-u` | `--undo` | 撤销模式：还原上一次操作并删除备份 | 0 | FR-014 |
| case_sensitive | `-c` | `--case-sensitive` | 大小写敏感（1=敏感, 0=不敏感） | 1 | FR-015 |
| regex | `-r` | `--regex` | 使用正则表达式（1=正则, 0=字面） | 0 | FR-016 |
| progress | | `--progress` | 显示进度（1=显示, 0=隐藏） | 1 | FR-017 |

### 参数组合规则

- **非撤销模式（`-u 0`）**: MUST 提供 `-o`/`-n` 或 `--rules` 之一；二者皆缺 → 非法输入。
- **同时提供 `-o/-n` 与 `--rules`**: 命令行单组规则追加到规则集（FR-007b）。
- **撤销模式（`-u 1`）**: `-o`/`-n`/`--rules` 均可省略（FR-014、spec 边界）。
- **全局开关**: `-r`、`-c` 对本次运行的所有规则统一生效（FR-016b）。

## 输入/输出约定

- 进度与诊断 → stderr（`--progress 1` 时显示 spinner + 计数）。
- 结束摘要 → stdout：被修改文件数、跳过二进制数、失败数、总替换次数、备份路径（若有）。FR-021。

## 退出码契约

| 情形 | 退出码 |
|------|--------|
| 成功完成（含"已修改 0 个文件"） | 0 |
| 目录不存在/不可访问 | 非 0 |
| 旧文本为空 / 规则文件非法 / 无有效规则 | 非 0 |
| 正则表达式无效 | 非 0 |
| 撤销模式下无匹配当前目录的备份 | 非 0（给出明确提示） |

对应 FR-022 与 spec 边界。

## 行为契约（关键语义）

1. **递归遍历**目标目录全部文件，对文本文件替换；**二进制自动跳过**（FR-008、FR-009）。
2. **单遍同时匹配**：多规则一次扫描，每位置至多命中一条，重叠按规则顺序优先，无级联（FR-008a）。
3. **正则捕获组**：`-r 1` 时 `-n` 支持 `$1`/`${name}`，未匹配组按空串（FR-016a）。
4. **备份**：`-b 1` 时仅对被改文件，在覆盖前写入 `<目标目录父目录>/backup/backup_yyyyMMddHHmmss.zip`，
   内部保留层级 + 来源目录清单（FR-011~FR-013a）。
5. **原子写回**：临时文件 + 原子改名，崩溃不留半写文件（FR-019a）。
6. **大文件**：超阈值文件流式重叠窗口替换，内存恒定（FR-019）。
7. **撤销**：`-u 1` 按目标目录定位最近备份，还原覆盖后删除该归档（FR-014）。

## 使用示例（契约级）

```bash
# 字面替换：将目录内所有文本文件的 foo 改为 bar
cct trt -d ./project -o foo -n bar

# 正则 + 捕获组：交换日期 yyyy-mm 为 mm/yyyy
cct trt -d ./logs -r 1 -o '(\d{4})-(\d{2})' -n '${2}/${1}'

# 批量规则文件，关闭备份
cct trt -d ./project --rules ./rules.tsv -b 0

# 大小写不敏感替换
cct trt -d ./docs -o hello -n hi -c 0

# 撤销 ./project 上一次替换
cct trt -d ./project -u 1
```

### `rules.tsv` 示例（制表符分隔，`#` 为注释）

```text
# 旧文本<TAB>新文本
foo	bar
baz	qux
old-name	new-name
```
